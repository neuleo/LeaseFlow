# Track Specification: Enhance Financial Projections and Dashboard Accuracy

## Description
This track focuses on refining the recently added financial projection features, ensuring the daily overview is accurate, and verifying that all contract-specific rates are correctly applied across various edge cases (e.g., extremely high or low mileage).

## Goals
- Validate the tiered excess/under-mileage calculations.
- Ensure the daily quota tracking is robust against irregular logging intervals.
- Improve the visual clarity of financial warnings on the dashboard.
